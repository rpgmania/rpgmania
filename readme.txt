# Daggerheart Fight Club

This is a simple tool designed to help Game Masters (GMs) track resources in the Daggerheart role-playing game. 

## Overview

Daggerheart Fight Club provides functionalities to manage action tokens, fear tokens, and a countdown timer. It's built using HTML, Tailwind CSS for styling, and Stimulus.js for interactivity.

### Features

- Track action tokens with increment and decrement buttons.
- Manage fear tokens with checkboxes, limited to a maximum of 10.
- Set and adjust a countdown timer with increase and decrease buttons.

## Usage

1. Clone the repository or download the HTML file.
2. Open the HTML file in your web browser.
3. Interact with the buttons and checkboxes to track resources and manage the countdown timer.

## Technologies Used

- HTML: Provides the structure and layout of the tool.
- Tailwind CSS: Used for styling the interface and components.
- Stimulus.js: Handles interactivity and dynamic behavior of the tool.
